# power-BI-study

### 売上分析ダッシュボード（Power BI）

#### プロジェクト概要

本プロジェクトは、Power BIを初めて使用して作成した売上分析レポートです。
データ分析の基礎を学ぶワークショップに参加し、データの取り込みからモデリング、ビジュアライゼーションまでの一連の流れを実践しました。


#### 学習背景

きっかけ: データの可視化に興味を持ち、初めてPower BIに触れました。

ワークショップでの学び:
散らばったデータのクレンジング（Power Query）

カンマ区切りや単位（K/M）の適切な書式設定
ユーザーが直感的に操作できるボタンスライサーの実装

データの意味を正しく伝えるためのグラフ選択（円グラフやカードビジュアル）


#### レポートのポイント

直感的な操作: 画面上部のボタンスライサーにより、月ごとの数値を瞬時に切り替えて確認できます。

視認性の向上: カードビジュアルの単位を日本語（売上高の合計など）に修正し、誰が見ても分かりやすい数値を表示しました。

デザインの統一感: 16:9の全画面表示に最適化し、背景色と各ビジュアルを調整してプロフェッショナルな見た目を意識しました。



#### 使用したツール・技術

Power BI Desktop

DAX: 基本的な集計メジャーの作成

GitHub: 成果物の公開とバージョン管理
